# Skobki
//ENG
This program checks the correct closing of various brackets: () or {} or [].
If some parenthesis is missing, the program will return FALSE.
//RUS
Данная программа проверяет строку на правильное закрытие различных скобок: () или {} или []. 
Если какой-то скобки не хватает программа выдаст FALSE.
